Welcome

Welcome to the NC State Policies, Regulations, and Rules (PRR) website. All PRRs have been organized into functional categories. You can use the “View All…” and “Select a Category” menus to find a PRR.

NCSU’s Procedure for Formatting, Adopting, and Publishing Policies, Regulations, and Rules (PRR Protocol) provides information about the formatting, adoption and publishing of PRRs. You may obtain more information on the distinctions between PRRs on the definitions page. The overview page provides information about the PRR website and how it operates.
