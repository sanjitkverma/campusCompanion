Electric Vehicles

NC State has Electric Charging Stations

NC State University welcomes employees and students using electric vehicles. We encourage their use to reduce campus air pollution and greenhouse gas emissions. We offer several charging stations across campus. Please see the location map to find your closest charging station.

All stations require a ChargePoint card or a ChargePoint Mobile App for your Smartphone.

With a Valid NC State Parking Permit

  • Hourly Use.
  • Submit a ChargePoint Connection Request to receive the $1/hour rate for the first 4 hours of connection time.
    • Please allow up to two business days for your request to be reviewed.
  • $2.50/hour without a ChargePoint Connection for the first 4 hours of connection time.
  • Vehicles exceeding the 4-hour connection time are subject to a $10/hour charge, prorated by the minute.

Without a Valid NC State Parking Permit

  • Hourly use.
  • $2.50/hour for the 4 hours of connection time.
  • Vehicles exceeding the 4-hour connection time are subject to a $10/hour charge, prorated by the minute.

How to Submit a ChargePoint Connection Request

Entering a Connection Code using the ChargePoint App:
  • In the ChargePoint app, go to:
    • Main Menu
    • Account
    • Connections
    • Browse Connections
  • Enter the NCSU connection code NCSUEV and click search.
  • Select NC State University when it appears on the next screen.
  • Next, answer the questions listed and submit your request to connect.
Entering a Connection Code using the ChargePoint Website:

  • Log in to the ChargePoint website www.chargepoint.com.
  • Click Connections from the menu at the top of the home screen.
  • Click Use Connection Code.
  • Enter the connection code NCSUEV in the search box on the right side of the page and click the Apply button.
  • When the table appears on the page, click Request on the right side of the row that says NC State University.
  • Next, answer the questions listed and submit your request to connect.
